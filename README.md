#  CodSoft Data Science Internship Projects

##  Overview

This repository contains the projects completed during my **CodSoft Data Science Internship**. These projects focus on applying machine learning techniques, data preprocessing, exploratory data analysis, and predictive modeling on real-world datasets.

Each task demonstrates a different aspect of data science and machine learning, helping build practical experience in solving data-driven problems.

---

##  Projects Included

###  Task 1: Titanic Survival Prediction

Developed a machine learning model to predict whether a passenger survived the Titanic disaster based on various features such as age, gender, passenger class, and fare.

####  Skills Applied
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Classification Algorithms
- Model Evaluation

---

###  Task 2: Iris Flower Classification

Built a classification model capable of identifying Iris flower species using sepal and petal measurements.

####  Skills Applied
- Data Visualization
- Data Preprocessing
- Supervised Learning
- Classification Techniques
- Performance Analysis

---

###  Task 3: Credit Card Fraud Detection

Created a fraud detection system to classify financial transactions as legitimate or fraudulent using machine learning algorithms.

####  Skills Applied
- Handling Imbalanced Datasets
- Data Preprocessing
- Classification Models
- Fraud Detection Techniques
- Model Evaluation Metrics
