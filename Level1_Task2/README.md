# Personal Portfolio Website

## Overview

This project is a Personal Portfolio Website developed using HTML and CSS as part of the Oasis Infobyte Web Development and Designing Internship (Level 1 - Task 2).

The portfolio website serves as a professional platform to showcase personal information, technical skills, projects, achievements, education, and contact details. It is designed with a clean, responsive, and visually appealing user interface to highlight an individual's profile and accomplishments.

---

## Features

- Responsive design for desktop and mobile devices
- Professional introduction section
- About Me section
- Skills showcase
- Projects section
- Education details
- Resume/CV section
- Contact information
- Social media links
- Modern and user-friendly interface

---

## Technologies Used

- HTML5
- CSS3
